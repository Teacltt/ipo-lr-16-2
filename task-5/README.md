# ipo-lr-17
На главной странице сайта есть ссылки на автора проекта и на сам магазин.
Когда вы переходите на страницу об авторе, то видите фамилию, имя автора, а также его цитату дня.
Когда перезодите на страниуц о магазине, то можете увидеть на ней информацию о возмможных покупках.
С обеих страниц можно перейти обратно на главную страницу сайта.
